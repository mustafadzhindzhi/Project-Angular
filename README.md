## WEB DESIGN STUDIO
Welcome to the official documentation for Web Design Studio. This platform serves as a curated hub for student designers to exhibit their innovative UI projects, while providing developers with a seamless experience to discover, evaluate, and integrate these projects into their own endeavors. Our mission is to facilitate a space where creativity thrives and tailored solutions meet the unique demands of modern design and development.

## Key Features
    1.Designer Showcase: Web Design Studio provides a dedicated space for student designers to flaunt their UI projects. Whether it's a sleek portfolio website, a cutting-edge mobile app interface, or a visually stunning web application, designers can exhibit their creations and gain recognition within the community.

    2. Customization Capabilities: Web Design Studio provides a simple form for designers to customize their projects. Upload a main photo, specify systems used, outline deliverables, describe your approach, share challenges faced, and upload additional images. Tailor your project to your specific vision and goals with ease.

    3. Community Interaction: At Web Design Studio, designers can engage with and draw inspiration from the broader community through likes. By liking projects that resonate with them, designers can show their appreciation, provide positive feedback, and support their peers. This simple yet effective interaction fosters a supportive and collaborative environment, encouraging designers to connect with one another and build meaningful relationships within the community.

    4. Educational Resources: Beyond project showcase, Web Design Studio also serves as an educational hub for aspiring designers. Through tutorials, articles, and case studies, designers can learn best practices, refine their skills, and stay abreast of the latest trends and technologies in the realm of UI/UX design.

Web Design Studio offers a platform for student designers to showcase their UI projects, with customization options allowing for tailored creations. Community interaction features, like likes, foster collaboration and support among designers. Additionally, the platform provides educational resources to help designers learn and grow in the field of UI/UX design.

## How to Access

Access to Web Design Studio's comprehensive features is designed to ensure an engaging and rewarding experience for all designers in our community. Here's how access is structured:

Non-Registered and Logged-Out Users: 
Visitors who have not registered or logged in will have limited access to the platform. They can browse public sections of the site, such as the homepage, what we do, about us and contacts, but won't be able to interact with content or access the full range of features available to registered members.

Registration Process: 
To unlock the full potential of Web Design Studio, designers are encouraged to register. The registration process is simple. You'll need to provide a username, email, profile picture, telephone number, and password. After successful registration, designers will receive a confirmation email, welcoming them to the Web Design Studio community.

## Benefits for Registered Users:
Access to Full Features: 
Registered users gain access to the complete range of features and functionalities of Web Design Studio, enabling them to fully utilize the platform and engage with its community.
Project Management: 
Registered users can efficiently manage their projects, including editing, deleting, and updating project details, ensuring their portfolio remains accurate and up-to-date.
Enhanced Profile: 
Registered users can personalize their profiles with a profile picture and additional details, helping them distinguish themselves within the community and foster connections with peers.
Community Engagement:
Registered users can interact with other designers, such as liking projects, fostering collaboration within the community.
Exclusive Content: 
Registered users may have access to exclusive resources or events, providing added value and incentive for registration.
Future Opportunities:
Being a registered user may open up future opportunities, such as collaboration requests, job offers, or participation in special projects or initiatives hosted by Web Design Studio.

## Client Application Setup

Clone the Repository: 
You can clone the repository using the following command or download it as a ZIP file and extract it on your computer.

Navigate to the Project Directory:
Use the terminal to navigate to the project directory.

cd my-project
Install Dependencies:
Install all the necessary dependencies by running the following command in your terminal:

npm install   
Run the Client Part: Start the Angular development server with this command:
ng s 
Open the Project: Access the application by opening the following URL in a web browser: http://localhost:4200/
Server part Setup

# Navigate to the Server Directory:
cd server
Install Server Dependencies and Start the Server Execute the following commands in order to start the server.
npm install
npm start
Running the Server:
Once the server is started, it will listen for requests on:

http://localhost:3030/
